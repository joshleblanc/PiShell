# Pi Configuration

This directory contains configuration for the pi coding agent:

- `AGENTS.md` - Main agent instructions
- `IDENTITY.md` - Bot identity and personality
- `SOUL.md` - Core values and behavior guidelines
- `MEMORY.md` - Persistent memories across sessions
- `BOOTSTRAP.md` - Initial session setup
- `HEARTBEAT.md` - Periodic status check prompts
- `extensions/` - Custom extensions
- `skills/` - Agent skills
- `prompts/` - Reusable prompt templates
- `sessions/` - Conversation history
